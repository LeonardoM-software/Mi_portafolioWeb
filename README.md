# Proyecto: Página Web de Comida Rápida

Proyecto académico desarrollado como parte de la formación en Desarrollo de Software.
Consiste en el diseño y desarrollo de una página web informativa para un negocio de comida rápida,
con el objetivo de presentar el menú, promociones y datos de contacto de forma clara y atractiva.

## Tecnologías utilizadas
- HTML
- CSS

## Funcionalidades
- Página de inicio informativa
- Sección de menú
- Sección de promociones
- Sección de contacto

## Rol y aportes personales
Desarrollo completo del proyecto, incluyendo la estructura del sitio web,
el diseño visual y la organización del contenido.
